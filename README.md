# 🚀 QR Safe – Digital Document & Item Management System

QR Safe is a full-stack web application built with **FastAPI (Dockerized Backend)** and **React (Frontend)** that allows users to securely manage personal documents, vehicles, and QR-based item tracking.

The system supports:

- Profile Management
- Personal Document Upload (Aadhaar, PAN, DL, etc.)
- Vehicle Management
- Secure Cloud Storage
- QR Code Generation
- Cloud-based Deployment

---

## 🏗️ Tech Stack

### 🔹 Backend
- FastAPI
- Python 3.10+
- Docker
- MongoDB Atlas
- Cloudinary (File Storage)
- Loguru (Logging)

### 🔹 Frontend
- React
- Axios
- Tailwind CSS

### 🔹 Deployment
- Railway (Backend)
- Vercel (Frontend)
- MongoDB Atlas (Database)
- Cloudinary (Storage CDN)
